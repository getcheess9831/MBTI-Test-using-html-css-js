# MBTI Personality Test

A simple and interactive web app that helps users discover their MBTI personality type by answering 12 questions.

## 🔗 Live Demo
[Click here to take the test](https://getcheess9831.github.io/MBTI-Test-using-html-css-js/)

## 📋 Features
- 12 questions covering all four MBTI dimensions (E/I, S/N, T/F, J/P)
- Instant personality type result upon submission
- Unique illustration for each of the 16 MBTI types
- Responsive design for mobile and desktop
- Input validation — prompts user to answer all questions before submitting

## 🛠️ Built With
- HTML
- CSS
- JavaScript
